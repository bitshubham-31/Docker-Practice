# 🐳 Docker Practice

This repository contains hands-on practice and examples to learn **Docker** from basics to intermediate level. It is designed to help beginners understand containerization and work with real-world Docker workflows.

---

## 🚀 What is Docker?

Docker is a containerization platform that allows you to package applications along with their dependencies into lightweight, portable containers that can run anywhere. :contentReference[oaicite:1]{index=1}


---

## 🎯 Objectives

- Understand Docker basics
- Learn how to build Docker images
- Run and manage containers
- Work with Dockerfile and Docker Compose
- Practice real-world DevOps workflows

---

## ⚙️ Prerequisites

Make sure you have installed:

- Docker
- Docker Compose (optional)

docker --version
